## 🆘 Need Help?

### Server Won't Start
- Make sure you're in the right directory
- Check Python is installed correctly
- Try: `python --version`

### Can't Login
- Clear browser cache/cookies
- Try different browser
- Check credentials are exact (including capitals)

### Panel Shows 404
- Make sure server is running
- Check URL exactly: `/appointments/pending/`
- Try refreshing the page

### Appointments Not Showing
- Create test appointments first
- Check admin panel at `/admin/core/appointment/`
- Try refresh button

## 🎯 API Usage Examples:

### Get Pending Appointments (Python)
```python
import urllib.request, json

req = urllib.request.Request(
    'http://localhost:8000/api/pending-appointments/',
    headers={'Authorization': 'Basic ...'}
)
response = urllib.request.urlopen(req)
data = json.loads(response.read())
print(data['appointments'])
```

### Update Appointment Status
```python
data = json.dumps({
    'appointment_id': 1,
    'status': 'Confirmed'
}).encode()

req = urllib.request.Request(
    'http://localhost:8000/api/pending-appointments/',
    data=data,
    headers={'Content-Type': 'application/json'}
)
response = urllib.request.urlopen(req)
```
